📗Excercise: Combine Two Tables
  . Difficulty: Easy
  . 🔗 Source: https://leetcode.com/problems/combine-two-tables/
  . Table Schema: 
    1. Person(personId, lastName, firstName)
    2. Address(addressId, personId, city, state)
  . 📄Problems: 
    - show first name, last name, city, state
    - include all person
    - if no address -> output null

✅ Solution:

  💡 Idea
  - We need all records from Person → use LEFT JOIN
  - Address contains personId (foreign key)
  - Match both tables using personId
  - If no match → Address fields become null

  ➡️ solution code: SQL
  ----------------------------------------------------
    SELECT p.firstName, p.lastName, a.city, a.state
    FROM Person as p
    left join Address as a 
    on p.personId = a.personId;
  ----------------------------------------------------

🍀What I learned:
  - using left join when you're interested in the left side table
  - left join will return null if the left record doesn't match the record in the right one
  - tables are joined using the FK (personId)
