📗Excercise: Second Highest Salary

  . Difficulty: Medium

  . 🔗 Source: https://leetcode.com/problems/second-highest-salary/

  . Table Schema: 

    1. Employee(id, salary)

  . 📄Problems: 

    - find the 2nd highest distinct salary (SecondHighestSalary)

    - if no 2nd highest -> return null
    

✅ Solution:

  💡 Idea
  - Use DISTINCT to remove duplicate salaries
  - order the salary in descending order
  - select the second one from that using limit 1 offset 1
  - Use IFNULL to return null if no 2nd highest salary exists

  ➡️ solution code: SQL
  ----------------------------------------------------
    select ifnull(
      (
        select distinct salary
        from Employee
        order by salary desc
        limit 1 offset 1
      ),
      null
    ) as SecondHighestSalary;
  ----------------------------------------------------

🍀What I learned:
  - order by desc: highest - lowest
  - using ifnull(value, "show what you want")
  - limit n -> n is the number of rows you want to show
  - offset m -> m is the number of rows to skip before starting
  - so limit 1 offset 1 returns the 2nd row
  - note: offset always work after sort so ORDER BY is very important
